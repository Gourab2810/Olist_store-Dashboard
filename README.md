# 🛒 Olist E-Commerce Performance Dashboard (Tableau)

## 📌 Project Overview

This project presents an interactive **E-Commerce Performance Dashboard** built using **Tableau**, based on the Olist dataset.
The dashboard provides deep insights into sales performance, customer behavior, payment trends, and delivery efficiency.

---

## 🎯 Objective

The main objective of this project is to:

* Analyze e-commerce sales performance
* Understand customer purchasing patterns
* Identify preferred payment methods
* Evaluate delivery performance and customer satisfaction

---

## 🛠️ Tools & Technologies

* Tableau
* Microsoft Excel (Data Cleaning & Preprocessing)

---

## 📊 Key KPIs

* **Total Orders:** 103.9K
* **Total Payments:** $16.0M
* **Total Revenue (Price):** $12.74M
* **Total Customers:** 96.5K
* **Total Sellers:** 102.4K

---

## 📈 Dashboard Insights

### 🔹 Sales & Order Analysis

* Majority of orders occur on **weekdays (78%)**, compared to weekends (22%)
* High order volume indicates strong platform engagement

### 🔹 Customer Satisfaction

* Most customers gave **5-star ratings**, showing high satisfaction
* Positive correlation between faster delivery and better reviews

### 🔹 Payment Trends

* **Credit Card** is the most preferred payment method
* Other methods like boleto and vouchers are used less frequently

### 🔹 Delivery Performance

* Average delivery time for pet shop category is around **11 days**
* Faster delivery leads to higher review scores

### 🔹 Geographic Insights

* Major customer concentration observed in **São Paulo**
* Regional analysis helps identify high-performing states
